CD1-9 show how i roughly intended the layers to fit together to make the scenes.
in each case, if there's a "dark" and "lite" version, it means crossfade from one to the other over time to create the scene.

Here's how it plays out:

start with the dark layers, to create CD1. note there are 2 layers of clouds and duplicate clouds as much as necessary to create a stormy sky. they're on "strings" so you can animate them jerking across the screen as if someone were playing a simple theater. that's the intent since there's no fancy anim here.

have the clouds sliding along the screen for a bit then switch to CD2 - the only diff is the lightning which comes in 2 layers, check you insert in the right order, they're labelled.

lightning only lasts a frame or two, then switch to CD3 using the special provided masks (lightning_flash_foreground/background)

that was one frame again, now back to CD2 (CD4 is the same as CD2) for a frame or two. then the lightning is gone.

CD5 now begins the crossfade from dark to lite versions for each layer. start removing some clouds off the screen so the sky clears, this results in CD8 eventually.

CD9 pops up the flower sprite. you could make this start small and expand, so it looks like it's growing. plonk em all over the place :) go wild! you end with CD9 (actually i edited slightly so the final version is cuter now)


The castle on the hill is designed so you can parallax it - just slide it to the left a bit when ready and you have space on the right side for a menu.
